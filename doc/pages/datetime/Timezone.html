function at(utc:DateTime):DateTime

Find out what was the date/time at specified UTC time in this timezone E.g. var utc = DateTime.fromString('2012-01-01 00:00:00'); var tz = Timezone.fromHHMM(400); tz.at(utc).toString() // 2014-01-01 04:00:00

function format(format:String, ?dt:DateTime = 0):String

Make a string according to format.

%z The time zone offset. Example: -0500 for US Eastern Time (TODO) %Z The time zone abbreviation. Example: EST for Eastern Time
